Guide d'utilisation

{% if user.is_admin %}Administrateur{% elif user.is_editor %}Éditeur{% else %}Lecteur{% endif %} — Palmarès Scolaires

Navigation générale

  • La barre latérale gauche donne accès à toutes les sections selon votre rôle.
  • La barre de recherche (page Élèves) accepte les fautes de frappe — tapez le nom approximatif.
  • Les notifications apparaissent en bas à droite après chaque action.

L Consultation & Exports {% if not user.is_reader %}(disponible pour tous){% endif %}

  • Rechercher un élève : menu Élèves → saisir le nom dans la barre de recherche.
  • Voir la fiche d'un élève : cliquer sur son nom → historique complet + graphique d'évolution.
  • Consulter les classements : menu Exports → choisir une année / section / classe → voir le palmarès.
  • Exporter en PDF ou CSV : menu Exports → sélectionner les filtres → cliquer sur le bouton de téléchargement.
{% if user.is_editor %}

É Édition & Imports {% if not user.is_admin %}(votre rôle){% endif %}

  • Importer un fichier Excel : menu Importations → Nouvel import → glisser-déposer le fichier → lancer la vérification → consulter le rapport → confirmer l'import.
  • Corriger une note : menu Résultats → choisir la classe → cliquer sur la note → modifier → sauvegarder. La vérification est réinitialisée automatiquement.
  • Valider un résultat : cliquer sur le badge gris À vérifier → il passe en vert Vérifié. Cliquer à nouveau pour annuler.
  • Modifier le nom d'un élève : fiche élève → cliquer sur le nom en haut → modifier en ligne → sauvegarder.
  • Fusionner des doublons : fiche élève → bouton Fusionner avec un autre élève → rechercher le doublon → confirmer. Tous les résultats sont regroupés sous un seul profil.
  • Annuler un import : menu Importations → trouver l'import → bouton Annuler (rollback). Toutes les notes créées par cet import sont supprimées.
  • Supprimer une note : fiche élève → icône corbeille sur la ligne concernée → confirmer.
  • Supprimer un élève : fiche élève → bouton Supprimer cet élève → confirmer. Toutes ses notes sont également supprimées.
{% endif %} {% if user.is_admin %}

A Administration (votre rôle)

  • Gérer les utilisateurs : menu Administration (lien vers l'admin Django) → section Accounts / Utilisateurs → créer, modifier, changer le rôle ou désactiver un compte.
  • Tableaux de bord globaux : menu Statistiques → KPIs, tendances, taux de réussite par section, palmarès des meilleurs élèves, activité par utilisateur.
  • Journal d'audit : menu Journal d'audit → chaque modification est tracée (qui, quand, valeur avant/après).
  • Purger des données : via l'admin Django → sélectionner les enregistrements → action Supprimer. Cette action est irréversible.
{% endif %}
